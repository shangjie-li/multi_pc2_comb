# multi_pc2_comb

ROS package for combining multiple pointcloud

## 安装
 - 建立工作空间并拷贝这个库
   ```Shell
   mkdir -p ros_ws/src
   cd ros_ws/src
   git clone https://github.com/shangjie-li/multi_pc2_comb.git
   cd ..
   catkin_make
   ```

## 运行
 - 启动`multi_pc2_comb`
   ```Shell
   roslaunch multi_pc2_comb multi_pc2_comb.launch
   ```
